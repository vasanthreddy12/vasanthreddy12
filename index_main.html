ABOUT THE PROJECT

During natural disasters, for giving updates, people use social media platforms such as twitter hugely. Social media is the most relevant source in the time of crisis situations. This posted information is very valuable for many humanitarian organizations. The posted data is available in various modalities like text, image, etcâ€¦ Using that information the humanitarian organizations gain the awareness of the situation in quick time to provide help and distribute aid resources according to the need of the resources. Using this information these organizations can spend their resources according to the need quickly, cost-effectively and easily to reduce the possibility of damage, injuries and even loss of lives. Therefore analyzing this posted data to get more information to humanitarian organizations is very helpful in crisis situations. There are various studies that have only analyzed the text data from the twitter posts. Clearly, there is a need to explore other modalities for getting more information in these crisis situations. In this study, we propose to use a multimodal approach using both text and image modalities of Twitter data to determine whether a tweet is informative for humanitarian aid and to further classify them based on the relevant information they provide such as reports of infrastructure damage, deceased people, and other categories.

Results

Our model achieves state-of-the-art performance on the CrisisMMD dataset with an accuracy of 93.3% for the Informative task which surpasses the baseline performance by 8.9% and an accuracy of 91.1% for the Humanitarian task which surpasses the baseline performance by 12.7%

Conclusion

We proposed a model architecture containing CLIP image and text encoders followed by align fusion, which achieve the state-of-the-art performance quickly in 20 epochs on CrisisMMD dataset. It has only 4 trainable layers which are: two single layer feed forward networks following CLIP image and text encoders, a single layer feed forward network following fusion and finally the softmax output layer.
